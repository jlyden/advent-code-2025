# Advent of Code 2025

https://adventofcode.com/2025

## Getting Started

```
git clone https://github.com/jlyden/advent-code-2025.git
cd advent-code-2025
npm install
```

## Run solutions

From terminal:
```
npm run build
node dist/day-one/solve-challenge.js <sample|input> <2>
```
