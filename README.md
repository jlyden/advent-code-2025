# Advent of Code 2025

https://adventofcode.com/2025

## Getting Started

```
git clone https://github.com/yourusername/advent-code-2025.git
cd advent-code-2025
npm install
```

## Run solutions

From terminal:
```
npm run build
node dist/day-one/solve-challenge-one.js <sample|input>
```
